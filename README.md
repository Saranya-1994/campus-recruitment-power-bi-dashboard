# 📊 Campus Recruitment Power BI Dashboard

## 📌 Project Overview
This project analyzes campus recruitment data to understand placement outcomes, salary trends, academic performance, and demographic patterns.

## 🛠 Tools Used
- Power BI
- Excel / CSV
- Data Modeling
- DAX Measures

## 📈 Key Insights
- Total students: 215
- Placement rate: ~69%
- Higher average salaries for Sci & Tech degrees
- Candidates with work experience earn higher salaries
- Gender distribution shows higher male participation
